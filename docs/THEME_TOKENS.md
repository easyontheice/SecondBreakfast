- Background: `#0E1410`
- Surface: `#131B14`
- Panel/Card: `#182219`
- Border: `#2A3A2B`
- Text: `#E9E3D6`
- Muted Text: `#B9B0A1`
- Primary (Brass): `#C6A15B`
- Primary Hover: `#D7B46D`
- Accent (Moss): `#6F8F5E`
- Danger: `#B25A4A`

## Interaction

- Focus ring: brass token (`--ring`)
- Buttons: tactile with warm borders and subtle inner highlights
- Cards: olive slate panel + warm top highlight
- Inputs: inset background with brass focus state
